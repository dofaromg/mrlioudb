## ✅ 部署驗證 (必須完成)

### 基礎檢查

- [ ] **叢集連接正常**
  ```bash
  kubectl cluster-info
  ```
  - 預期: 能看到叢集資訊和 API server 地址

- [ ] **所有命名空間已建立**
  ```bash
  kubectl get namespaces | grep -E "flowagent|monitoring"
  ```
  - 預期: 看到 `flowagent` 和 `monitoring` 命名空間

- [ ] **所有 Deployments 就緒**
  ```bash
  kubectl get deployments -n flowagent
  ```
  - 預期: MongoDB (1/1), Module-A (2/2), Orchestrator (1/1), Next.js Frontend (2/2)

- [ ] **所有 Pods 運行中**
  ```bash
  kubectl get pods -n flowagent
  ```
  - 預期: 所有 Pods 狀態為 `Running`，READY 為 `1/1`

- [ ] **所有 Services 已建立**
  ```bash
  kubectl get svc -n flowagent
  ```
  - 預期: 看到 4 個服務，LoadBalancer 服務有外部 IP

- [ ] **持久化存儲已綁定**
  ```bash
  kubectl get pvc -n flowagent
  ```
  - 預期: MongoDB PVC 狀態為 `Bound`

## 🔍 功能測試

### 服務可訪問性

- [ ] **Next.js Frontend 可訪問**
  ```bash
  FRONTEND_IP=$(kubectl get svc nextjs-frontend -n flowagent -o jsonpath='{.status.loadBalancer.ingress[0].ip}')
  curl -I http://$FRONTEND_IP
  ```
  - 預期: HTTP 200 或 301/302
  - 實際 IP: _______________

- [ ] **Orchestrator 可訪問**
  ```bash
  ORCH_IP=$(kubectl get svc orchestrator -n flowagent -o jsonpath='{.status.loadBalancer.ingress[0].ip}')
  curl -I http://$ORCH_IP
  ```
  - 預期: HTTP 200
  - 實際 IP: _______________

- [ ] **內部服務連通性**
  ```bash
  kubectl run test-pod --rm -i --tty --image=busybox -n flowagent -- wget -O- http://module-a:8080
  ```
  - 預期: 能成功連接

### 日誌檢查

- [ ] **MongoDB 日誌正常**
  ```bash
  kubectl logs deployment/mongodb -n flowagent --tail=50
  ```
  - 預期: 沒有錯誤訊息，顯示 "waiting for connections"

- [ ] **Module-A 日誌正常**
  ```bash
  kubectl logs deployment/module-a -n flowagent --tail=50
  ```
  - 預期: 應用程式成功啟動，沒有致命錯誤

- [ ] **Orchestrator 日誌正常**
  ```bash
  kubectl logs deployment/orchestrator -n flowagent --tail=50
  ```
  - 預期: 服務運行正常

- [ ] **Next.js Frontend 日誌正常**
  ```bash
  kubectl logs deployment/nextjs-frontend -n flowagent --tail=50
  ```
  - 預期: Next.js 伺服器成功啟動

### 資源狀態

- [ ] **沒有警告事件**
  ```bash
  kubectl get events -n flowagent --field-selector type=Warning
  ```
  - 預期: 沒有或只有少數無害的警告

- [ ] **資源使用正常**
  ```bash
  kubectl top pods -n flowagent
  ```
  - 預期: CPU 和記憶體使用率在合理範圍
